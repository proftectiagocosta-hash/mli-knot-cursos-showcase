# Overview / Visão geral

This document summarizes the purpose of MLI-Knot-Cursos Showcase.

Este documento resume o propósito do MLI-Knot-Cursos Showcase.

---

## What this repository is / O que este repositório é

MLI-Knot-Cursos Showcase is a public sanitized documentation repository.

It presents the planned educational flow of the private MLI-Knot-Cursos project without exposing internal code or private development material.

O MLI-Knot-Cursos Showcase é um repositório público e sanitizado de documentação.

Ele apresenta o fluxo educacional planejado do projeto privado MLI-Knot-Cursos sem expor código interno ou material privado de desenvolvimento.

---

## Main idea / Ideia principal

The project is organized around a simple learning journey:

catalog -> enrollment -> lessons -> progress -> continue studying -> certificate

catálogo -> inscrição -> aulas -> progresso -> continuar estudo -> certificado

---

## Public value / Valor público

This showcase helps explain:

- how the course platform is intended to work;
- how learner progress is represented;
- how certificate availability is conceptually connected to completion;
- what is public, private, fictional, or not yet implemented;
- how the project can be presented safely before opening source code.

Esta vitrine ajuda a explicar:

- como a plataforma de cursos deve funcionar;
- como o progresso do estudante é representado;
- como a disponibilidade do certificado se conecta conceitualmente à conclusão;
- o que é público, privado, fictício ou ainda não implementado;
- como o projeto pode ser apresentado com segurança antes de abrir código-fonte.
