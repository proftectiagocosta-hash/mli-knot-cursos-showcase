# Certificate flow / Fluxo de certificado

This document describes the public sanitized certificate structure planned for **MLI-Knot-Cursos**.

Este documento descreve a estrutura pública e sanitizada de certificado planejada para o **MLI-Knot-Cursos**.

---

## Purpose / Propósito

The certificate flow represents the final step of the learning journey.

O fluxo de certificado representa a etapa final da jornada de estudo.

In the current showcase, this is a conceptual structure only.

Na vitrine atual, isso é apenas uma estrutura conceitual.

---

## Intended condition / Condição pretendida

A certificate structure should only become available after the learner reaches 100% course completion.

A estrutura de certificado deve ficar disponível apenas quando o estudante atingir 100% de conclusão do curso.

```text
course progress = 100% -> certificate action enabled
```

```text
progresso do curso = 100% -> ação de certificado liberada
```

---

## Public fields / Campos públicos

A safe public certificate preview may show:

- learner display name as placeholder;
- course title;
- completion status;
- completion date as placeholder;
- issuer/project name;
- non-official prototype notice.

Uma prévia pública segura de certificado pode exibir:

- nome fictício ou placeholder do estudante;
- título do curso;
- status de conclusão;
- data fictícia ou placeholder;
- nome do emissor/projeto;
- aviso de protótipo não oficial.

---

## Current boundaries / Limites atuais

This showcase does not represent official certificate issuance.

Esta vitrine não representa emissão oficial de certificado.

It does not include:

- official validation;
- real student identity;
- legal certificate status;
- payment validation;
- external registry;
- production certificate backend.

Ela não inclui:

- validação oficial;
- identidade real de estudante;
- status legal de certificado;
- validação de pagamento;
- registro externo;
- backend de certificado em produção.
