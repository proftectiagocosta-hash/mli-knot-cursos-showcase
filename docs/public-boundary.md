# Public boundary / Limite público

This document defines what belongs and what does not belong in this showcase repository.

Este documento define o que pertence e o que não pertence a este repositório de vitrine.

---

## Allowed here / Permitido aqui

This repository may include:

- public documentation;
- sanitized flow descriptions;
- fictional examples;
- placeholder names;
- public-safe screenshots;
- conceptual diagrams;
- roadmap notes;
- explanations of current limitations.

Este repositório pode incluir:

- documentação pública;
- descrições sanitizadas de fluxo;
- exemplos fictícios;
- nomes placeholder;
- capturas públicas seguras;
- diagramas conceituais;
- notas de roadmap;
- explicações sobre limitações atuais.

---

## Not allowed here / Não permitido aqui

This repository must not include:

- private source code from the main MLI-Knot-Cursos repository;
- real student data;
- real user accounts;
- credentials;
- API keys;
- payment information;
- private course content;
- unrevised internal notes;
- production backend details;
- official certificate records.

Este repositório não deve incluir:

- código-fonte privado do repositório principal MLI-Knot-Cursos;
- dados reais de alunos;
- contas reais de usuários;
- credenciais;
- chaves de API;
- informações de pagamento;
- conteúdo privado de cursos;
- notas internas não revisadas;
- detalhes de backend de produção;
- registros oficiais de certificados.

---

## Review rule / Regra de revisão

Before adding anything to this repository, check whether the content is:

public
sanitized
fictional when needed
safe to share
useful for explaining the project

Antes de adicionar qualquer conteúdo a este repositório, verifique se o conteúdo é:

público
sanitizado
fictício quando necessário
seguro para compartilhar
útil para explicar o projeto

If there is doubt, keep the content private.

Se houver dúvida, mantenha o conteúdo privado.
